# AnisOS
Made with HTML, CSS and  JavaScript (no, no frameworks or tailwind), this is basically my personal portfolio but make it look like a WebOS (just to be a bit cooler i guess, plus i loved doing the frontend). 

It has draggable windows, a bottom dock with mini apps (spotify, pixel art, a notes/to-do app and a calculator) and stickers you can move around the screen that open different windows. There is also a top bar with the time, date, weather and system info, 

The bes part, in my opinion, is the reactive mosaic background that lights up when you hover it (and it's pink too!). 

To be honest, the hardest part was getting my "Space Jump" game to load inside one of the windows. The main problem was that the game was already a project by itself so everything was conflicting, git took is as a submodule and there was a point where i didn't even know what to do, the whole style was off, the functions didn't function, and the event listeners did not listen at all. I fixed it though so everything is okay now, i just let the iframe do the reloading on open/close instead of trying to control it from outside. anyway i'm pretty happy with how it turned out!!
